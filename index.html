JavaScript

ภาษาสำหรับพัฒนาเว็บไซต์ให้มีความสามารถและโต้ตอบกับผู้ใช้งาน

1. JavaScript คืออะไร

ความหมาย

JavaScript (JS) คือ ภาษาสำหรับเขียนโปรแกรมที่ใช้เพิ่มความสามารถ ให้กับเว็บไซต์ เช่น ปุ่มกด การตรวจสอบข้อมูล แอนิเมชัน และการตอบสนองต่อผู้ใช้งาน

    ประวัติความเป็นมา

  • สร้างขึ้นในปี 1995
  • ผู้สร้างคือ Brendan Eich
  • พัฒนาที่บริษัท Netscape
  • เดิมชื่อ Mocha
  • เปลี่ยนเป็น LiveScript
  • สุดท้ายเปลี่ยนเป็น JavaScript

ความสำคัญ

  • ทำให้เว็บไซต์โต้ตอบกับผู้ใช้
  • ตรวจสอบข้อมูลก่อนส่ง
  • สร้าง Animation
  • เชื่อมต่อ API
  • ใช้พัฒนาเว็บไซต์และแอปพลิเคชัน

2. การทำงานของ JavaScript

Client-side

JavaScript จะทำงานบน Web Browser ของผู้ใช้งาน ทำให้เว็บไซต์ตอบสนองได้อย่างรวดเร็ว

ทำงานร่วมกับ HTML และ CSS

ภาษา หน้าที่
HTML โครงสร้างเว็บไซต์
CSS ตกแต่งเว็บไซต์
JavaScript เพิ่มความสามารถ
